Guia rápido de operações comuns no IRC:

1. Comandos comuns

  • /nick NOME - Altera o nickname para NOME. Lembre-se de que se o nickname já estiver registrado o servidor vai trocar automaticamente caso o mesmo não seja identificado a tempo. Leia mais em "Registro de usuário".
  • /users - Lista os usuários do canal.
  • /msg NOME MENSAGEM - Envia MENSAGEM para o usuário NOME. Somente este usuário receberá a mensagem (e quem estiver olhando para o monitor dele(a)).
  • NOME: MENSAGEM - Envia MENSAGEM para o canal (todo mundo do canal irá ver), porém faz a mensagem mudar de cor para o usuário NOME e emite um alerta sonoro (essa configuração depende do cliente que o usuário NOME estiver usando, alguns usuários desabilitam o som ou outros recursos).
  • AUTOCOMPLETAR - Quando começar a digitar o nome de um usuário ele será autocompletado pressionando o botão de tabulação. Neste cliente qwebirc (o webchat acima) isso só funciona com a primeira palavra da mensagem. Inclusive o cliente já irá completar com dois pontos e um espaço para mandar uma mensagem pública direta para o usuário em questão.
  • /join CANAL - Entra no canal CANAL. Segundo as especificações do irc todos canais começam com #. Por exemplo, para entrar no canal #freenode digite /join #freenode

2. Mensagens diretas privadas e públicas:

  • As mensagens que os outros mandam ficam em uma aba na parte superior do webchat com o respectivo nome da pessoa.
  • Novas mensagens fazem o texto da aba mudar para vermelho. Clique na aba com nome da pessoa para ler ou enviar mensagens privadas diretamente.
  • Para voltar para o canal clique na aba #freenode.
  • Quando alguém te manda uma mensagem direta no canal ela aparece com o teu nick seguido de dois pontos e de cor diferente. Se o flash estiver habilitado também será emitido um aviso sonoro. Se achar ruim tire o volume do frame, da aba do navegador, do navegador inteiro, da placa de som do sistema, do sistema inteiro, do computador inteiro ou do mundo inteiro, dependendo do caso. Ou desabilite o flash que também resolve.
  • Existe um usuário especial chamado NickServ que é um bot que registra nomes de usuário e apelidos. Veja abaixo na parte "Registro de usuário" como fazer para utilizar este serviço.

2.1. Como mandar mensagens privadas

  • Para enviar mensagens privadas, digite /msg NOME MENSAGEM onde NOME é o nickname da pessoa (como aparece na lista de usuários) e MENSAGEM é a mensagem que aparecerá como mensagem privada para o usuário. Cuidado pra não errar o comando (não esqueça da barra /) pra não mandar a mensagem pra todo canal! Quando estiver em uma janela de mensagem privada (nas abas com o nome do usuário em vez do nome do canal) não é necessário usar o comando /msg. Apenas envie a mensagem que somente aquele usuário receberá.

2.2. Como mandar mensagens públicas

  • Para enviar mensagens públicas para uma pessoa, digite o nome da pessoa seguido de dois pontos e um espaço. Se a pessoa tiver flash habilitado ouvirá um aviso sonoro indicando que alguém mandou mensagem para ela no canal. Por exemplo, para acordar um usuário dormindo na frente do computador envie diversas vezes usuario: para o canal. Esta mensagem aparecerá para todos usuários que estiverem no canal.

3. Registro de usuário:

  • Se o nome for alterado automaticamente é porque já existe esse nome de usuário cadastrado no servidor freenode.
  • Registrar o nome de usuário garante que ninguém consiga entrar nos canais com aquele nome específico. Também dá ao usuário registrado o poder de se tornar operador e até criar seus próprios canais.

3.1. Como se registrar

  • Para registrar um usuário, deve-se enviar mensagens ao usuário NickServ. Para enviar mensagem para um usuário leia acima na seção "Mensagens diretas privadas e públicas". O usuário para o qual a gente quer enviar mensagem é o NickServ, a mensagem é irrelevante, mas vamos mandar um comando que ele entende pra garantir que ele vai responder alguma coisa. Digite /msg NickServ HELP. Vai abrir uma aba com o nome NickServ e já com a resposta que é o retorno do comando HELP. Esse comando explica como fazer as coisas, mas se tu não entender inglês continue lendo essa explicação mesmo.
  • Agora NA JANELA DO NICKSERV envie a mensagem REGISTER senha e-mail obviamente trocando "senha" pela senha desejada para o cadastro e "e-mail" por um endereço de e-mail válido. O nickname registrado é o nickname atual. Abra o e-mail enviado pela freenode para completar o registro. Lá no e-mail tem a outra mensagem de confirmação que deve ser enviada para o usuário NickServ.
  • Agora quando for entrar de novo marque a caixa "Auth to services" antes de conectar e bote o nome de usuário e senha registrados nas caixas novas que vão aparecer. "Username" e "Nickname" não são a mesma coisa, mas provavelmente fazendo esse registro eles vão ser iguais.

3.2. Identificação manual

  • A identificação automática funciona no webchat, mas em outros clientes pode ser necessário se autenticar manualmente. O usuário NickServ vai detectar que o teu usuário entrou e vai solicitar identificação imediata caso contrário ele troca o nome. Isso é ótimo porque ninguém consegue acessar os canais usando o teu nome de usuário.
  • Mas como quem fez o registro foi tu e tu sabe a senha então vamos provar para o NickServ isso. É só mandar pra ele a mensagem IDENTIFY senha obviamente trocando "senha" pela senha atual. Caso esteja com o nick errado ou queira autenticar outro usuário use IDENTIFY nome senha trocando "nome" pelo nickname correto.
